– didActionCalled:type:source:

デリゲートへアクションの発生を通知します。

- (void)didActionCalled:(BCLAction *)action type:(NSString *)type source:(id)source

Parameters

action

発生したアクションの情報
type

アクションのタイプを示す文字列　ビーコン検知での発生の場合"Beacon"が、リージョン進入による場合"RegionIn"が、リージョン退出の場合"RegionOut"がセットされます。
source

アクションの発生要因　ビーコン検知で発生した場合ビーコンの情報がBCLBeacon型で、リージョン進入・退出で発生した場合リージョンの情報がBCLRegion型で格納されます。
